FIVE (5) HTML TAGS

blockquote
- Defines a section that is quoted from another source. It also use for large quotes that require a citation.
head
- Contains metadata/information for the document.
article
- Use for independent or self-contained content and it should make sense on its own, and it should be possible to distribute it independently from the rest of the website.
strong
- Defines strong important text (bold)
del
- Strikethrough the text
